# Orginizer

PowerShell tooling for provisioning the Project Organizer launcher and seeding new survey projects.

## Requirements
- Windows with administrative rights to install App Installer
- PowerShell 7.5 (installed automatically via winget or downloaded portable build)

## Setup
Run `_Organizer/Setup-Organizer.ps1` to install PowerShell 7.5 (if needed), deploy the Organizer launcher, and create a Start Menu shortcut.

## Usage
Launch `Project Organizer` from the Start Menu to start new projects using the guided form.
